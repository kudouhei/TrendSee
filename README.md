# TrendSee — 数据故事博主自动化运营系统

> 采集 → 分析 → 生成 的可持续运转系统，支持「数据故事博主」用可视化讲有趣的社会/商业洞察。

## 技术栈

| 层         | 技术                                              |
|-----------|---------------------------------------------------|
| 前端       | React 18 + TypeScript + Vite + Tailwind CSS        |
| 数据可视化  | Recharts                                          |
| 后端       | Python FastAPI + SQLAlchemy (async)               |
| 任务队列   | Celery + Redis                                    |
| 数据库     | SQLite (开发) / PostgreSQL (生产)                  |
| AI 分析    | OpenAI GPT-4o                                     |
| 爬虫       | Playwright (XHS/抖音) + PRAW (Reddit) + pytrends   |
| NLP        | jieba + VADER + 自定义情感词典                     |

## 快速开始

### 本地开发（推荐）

```bash
# 1. 克隆并配置环境变量
cp .env.example .env
# 编辑 .env，填入 OPENAI_API_KEY

# 2. 启动 Redis（需提前安装）
redis-server &

# 3. 一键启动所有服务
./scripts/start_dev.sh
```

访问：
- **前端**: http://localhost:5173
- **API Docs**: http://localhost:8000/docs
- **Celery 监控**: http://localhost:5555 (需另启动 flower)

### Docker 启动

```bash
cp .env.example .env
# 填入 API keys 后：
docker-compose up --build
```

## 四大核心模块

### 📡 模块1：趋势雷达
**目的**：每周/每月跨平台抓取趋势信号，在内容爆发前发现上升期话题

**输入**：关键词列表 + 平台选择  
**输出**：话题热力榜 + 生命周期分布 + AI 解读 + 平台分布图

### ⛏️ 模块2：评论区挖矿
**目的**：深挖特定话题下的受众真实情绪和隐藏观点

**输入**：话题词  
**输出**：情绪分布图 + 高频词云 + AI 评论洞察 + 创作机会点

### 🔬 模块3：爆款解剖室
**目的**：反向拆解高病毒传播内容，提炼可复用的爆款公式

**输入**：话题词  
**输出**：爆款 TOP 榜 + 标题模式分析 + AI 爆款公式 + 复制技巧

### 🎯 模块4：垂直精分
**目的**：针对特定行业/赛道做深度研究报告（每月 1-2 期）

**输入**：垂直领域 + 子话题列表  
**输出**：深度行业报告 + 公众号长文 + 视频脚本大纲

## 自动化内容生产

每个模块运行后可一键生成：

| 平台     | 格式                              |
|---------|-----------------------------------|
| 小红书   | 图文帖（emoji 风格，800-1200字）   |
| 公众号   | 深度长文（2500-3500字，含数据分析） |
| 视频脚本 | 5-8分钟结构化大纲                  |

## 数据采集说明

| 平台         | 方案                    | 备注                          |
|-------------|-------------------------|------------------------------|
| 小红书       | Playwright 无头浏览器   | 建议配置 Cookie 提升稳定性    |
| 抖音         | Playwright 无头浏览器   | 抓取公开热榜，建议配置 Cookie  |
| Reddit       | PRAW 官方 API           | 无 Key 时降级为公开 JSON API  |
| Google Trends| pytrends 库             | 无需密钥，注意请求频率         |

## 定时任务调度

| 任务           | 频率        |
|---------------|-------------|
| 趋势雷达（周报）| 每周一 08:00 |
| 抖音热榜采集   | 每天 09:00   |
| Reddit 趋势   | 每 6 小时    |
| Google Trends | 每天 07:00   |
